<h1>Whono</h1>
<hr>
<p>This is a game of Uno designed to be played in the browser.</p>
<p>
    It features a web client written in Rust that compiles to web assembly<br>
    and a web server for hosting the application and managing the lobby.<br>
    Connection between client and server are currently handled by websocket<br>
    with the eventual plan to support WebRTC for a more P2P system.
</p>
